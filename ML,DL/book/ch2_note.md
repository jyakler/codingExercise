## 퍼셉트론 알고리즘 요약  
	1. 가중치를 0 또는 랜덤한 작은 값으로 초기화  
	2. 각 훈련 샘플 x에서 다음 작업을 함  
			-출력 값  y 계산  
			-가중치 업데이트  
	
* 두 클래스가 선형적으로 구분되고 학습률이 충분히 작을때 수렴이 보장


아달린 - 퍼셉트론 차이점  
퍼셉트론 - 단위 계단 함수 사용  
아달린 - 선형 활성화 함수 사용

** 활성화함수 = 입력신호의 총합을 출력신호로 변환하는 함수
